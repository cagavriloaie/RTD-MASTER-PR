1. DESCRIPCIÓN GENERAL
================================================================================

RTD Master Pro es una aplicación de escritorio de grado profesional diseñada para
mediciones precisas de temperatura utilizando tanto sensores RTD (Detector de
Temperatura por Resistencia) como termopares, de acuerdo con múltiples estándares
internacionales.

SENSORES RTD:
Dispositivos de medición de temperatura de precisión que explotan el cambio
predecible en la resistencia eléctrica de metales puros con la temperatura.

TERMOPARES:
Sensores de temperatura que generan voltaje eléctrico basado en el efecto Seebeck,
ideales para temperaturas extremas y respuesta rápida.

La aplicación implementa ecuaciones estandarizadas y coeficientes definidos por
organizaciones internacionales de metrología (IEC, NIST, ASTM, GOST).

Los sensores RTD son dispositivos de medición de temperatura de precisión que
explotan el cambio predecible en la resistencia eléctrica de metales puros con
la temperatura. Esta aplicación implementa las ecuaciones estandarizadas y
coeficientes definidos por organizaciones internacionales de metrología.

4. MÉTODOS DE CÁLCULO Y ECUACIONES
================================================================================

4.1 CÁLCULO RTD DE PLATINO - Ecuación de Callendar-Van Dusen
-------------------------------------------------------------

La ecuación de Callendar-Van Dusen es el método internacionalmente aceptado para
calcular la resistencia RTD de platino en función de la temperatura.

Para T ≥ 0°C:
  R(T) = R₀(1 + AT + BT²)

Para T < 0°C:
  R(T) = R₀[1 + AT + BT² + C(T - 100)T³]

Donde:
  R(T) = Resistencia a temperatura T (en Ω)
  R₀   = Resistencia nominal a 0°C (en Ω)
  T    = Temperatura (en °C)
  A, B, C = Coeficientes de Callendar-Van Dusen

VALORES DE COEFICIENTES:

Pt385 (IEC 60751, ASTM E1137):
  A =  3.9083 × 10⁻³  °C⁻¹
  B = -5.775  × 10⁻⁷  °C⁻²
  C = -4.183  × 10⁻¹² °C⁻⁴

Pt391 (GOST 6651):
  A =  3.96847 × 10⁻³  °C⁻¹
  B = -5.847   × 10⁻⁷  °C⁻²
  C = -4.35    × 10⁻¹² °C⁻⁴

Pt392 (Estadounidense heredado):
  A =  3.9848 × 10⁻³  °C⁻¹
  B = -5.870  × 10⁻⁷  °C⁻²
  C = -4.40   × 10⁻¹² °C⁻⁴


CÁLCULO INVERSO (Resistencia → Temperatura):

Para R ≥ R₀ (T ≥ 0°C):
  Resolver ecuación cuadrática: BT² + AT + (1 - R/R₀) = 0

  T = (-A + √(A² - 4B(1 - R/R₀))) / (2B)

Para R < R₀ (T < 0°C):
  Resolver ecuación cúbica iterativamente usando métodos numéricos (Newton-Raphson)


4.2 CÁLCULO RTD DE COBRE Y NÍQUEL - Aproximación Lineal
--------------------------------------------------------

Los RTDs de cobre y níquel utilizan una relación lineal simplificada debido a sus
rangos de temperatura más estrechos y diferentes propiedades físicas.

Cálculo Directo (Temperatura → Resistencia):
  R(T) = R₀(1 + αT)

Cálculo Inverso (Resistencia → Temperatura):
  T = (R - R₀) / (R₀ × α)

Donde:
  R(T) = Resistencia a temperatura T (en Ω)
  R₀   = Resistencia nominal a 0°C (en Ω)
  α    = Coeficiente de temperatura de resistencia (en Ω/Ω/°C)
  T    = Temperatura (en °C)


COEFICIENTES DE COBRE:
  α₄₂₆ = 0.00426 Ω/Ω/°C
  α₄₂₇ = 0.00427 Ω/Ω/°C
  α₄₂₈ = 0.00428 Ω/Ω/°C

COEFICIENTES DE NÍQUEL:
  α₆₁₇ = 0.00617 Ω/Ω/°C (Ni100)
  α₆₁₈ = 0.00618 Ω/Ω/°C (Ni1000)
  α₆₇₂ = 0.00672 Ω/Ω/°C (Ni120)

5. ESPECIFICACIONES DE TOLERANCIA
================================================================================

5.1 CLASES DE TOLERANCIA IEC 60751 (RTDs de Platino)
-----------------------------------------------------

Clase AA (Máxima Precisión):
  Tolerancia = ±(0.1 + 0.0017|t|) °C

  Ejemplos:
    A   0°C: ±0.10°C
    A  50°C: ±0.185°C
    A 100°C: ±0.27°C
    A 200°C: ±0.44°C

Clase A (Precisión Estándar):
  Tolerancia = ±(0.15 + 0.002|t|) °C

  Ejemplos:
    A   0°C: ±0.15°C
    A  50°C: ±0.25°C
    A 100°C: ±0.35°C
    A 200°C: ±0.55°C

Clase B (Uso General):
  Tolerancia = ±(0.3 + 0.005|t|) °C

  Ejemplos:
    A   0°C: ±0.30°C
    A  50°C: ±0.55°C
    A 100°C: ±0.80°C
    A 200°C: ±1.30°C

Clase C (Menor Precisión):
  Tolerancia = ±(0.6 + 0.01|t|) °C

  Ejemplos:
    A   0°C: ±0.60°C
    A  50°C: ±1.10°C
    A 100°C: ±1.60°C
    A 200°C: ±2.60°C

Donde |t| es el valor absoluto de la temperatura en °C


5.2 TOLERANCIA DE RESISTENCIA A 0°C
------------------------------------

Para sensores Pt100:
  Clase AA: ±0.04 Ω a 0°C
  Clase A:  ±0.06 Ω a 0°C
  Clase B:  ±0.12 Ω a 0°C
  Clase C:  ±0.24 Ω a 0°C

10. TERMOPARES - DESCRIPCIÓN GENERAL
-------------------------------------

10.1 ¿QUÉ SON LOS TERMOPARES?
------------------------------
Un termopar es un sensor de temperatura que genera un voltaje eléctrico
(fuerza electromotriz termoeléctrica) proporcional a la diferencia de temperatura
entre dos uniones de metales diferentes. Se basa en el efecto Seebeck,
descubierto en 1821 por Thomas Johann Seebeck.

Principio de operación:
- Dos cables de metales diferentes se unen en un extremo (unión de medición)
- El otro extremo está a una temperatura de referencia (unión fría)
- La diferencia de temperatura genera un pequeño voltaje (µV-mV)
- El voltaje es directamente proporcional a la temperatura medida


10.2 COMPARACIÓN RTD vs TERMOPAR
---------------------------------

│ Característica         │ RTD                     │ Termopar                │

11. TIPOS Y ESPECIFICACIONES DE TERMOPARES
-----------------------------------------

11.1 TIPO K (CHROMEL-ALUMEL)
----------------------------
El tipo de termopar más común para uso general e industrial.

Materiales:
  Positivo (+): Chromel (90% Níquel, 10% Cromo)
  Negativo (-): Alumel (95% Níquel, 2% Manganeso, 2% Aluminio, 1% Silicio)

Especificaciones:
  Rango de temperatura: -270°C a +1372°C
  Sensibilidad a 0°C: ~41 µV/°C
  Código de color: Amarillo
  Estándar: IEC 60584, ASTM E230

Aplicaciones:
  - Industrial de uso general
  - Sistemas HVAC
  - Control de procesos
  - Hornos y calderas

Ventajas:
  + Amplio rango de temperatura
  + Bajo costo
  + Buena resistencia a la oxidación
  + Ampliamente disponible

Desventajas:
  - Deriva magnética por encima de 800°C
  - No apto para atmósferas reductoras
  - Precisión moderada


11.2 TIPO J (HIERRO-CONSTANTÁN)
--------------------------------
Termopar para la industria del hierro/acero y atmósferas reductoras.

Materiales:
  Positivo (+): Hierro (Fe)
  Negativo (-): Constantán (55% Cobre, 45% Níquel)

Especificaciones:
  Rango de temperatura: -210°C a +1200°C
  Sensibilidad a 0°C: ~52 µV/°C (la más alta)
  Código de color: Negro
  Estándar: IEC 60584, ASTM E230

Aplicaciones:
  - Industria del hierro y acero
  - Atmósferas reductoras
  - Aplicaciones con presupuesto limitado
  - Equipos heredados

Ventajas:
  + Alta sensibilidad
  + Muy bajo costo
  + Bueno para temperaturas medias

Desventajas:
  - Oxidación rápida por encima de 540°C
  - No para atmósferas oxidantes
  - Rango más limitado que Tipo K


11.3 TIPO T (COBRE-CONSTANTÁN)

-----------------------------
Termopar moderno con estabilidad mejorada a altas temperaturas.

  Rango de temperatura: -270°C a +1300°C
  Sensibilidad a 0°C: ~27 µV/°C
  Código de color: Naranja
  Estándar: IEC 60584

Aplicaciones:
  - Reemplazo del Tipo K (mejor estabilidad)
  - Altas temperaturas
  - Atmósferas oxidantes

Ventajas:
  + Estabilidad superior al Tipo K
  + Sin deriva magnética
  + Excelente resistencia a la oxidación

Desventajas:
  - Mayor costo que el Tipo K
  - Menor sensibilidad


11.6 TIPO S, R, B (METALES NOBLES - PLATINO)
---------------------------------------------
Termopares de platino para temperaturas muy altas y máxima precisión.

TIPO S: Platino-Platino/Rodio 10%
  Rango: -50°C a +1768°C
  Sensibilidad: ~6 µV/°C
  Aplicaciones: Farmacéutica, biotecnología, estándares de laboratorio

TIPO R: Platino-Platino/Rodio 13%
  Rango: -50°C a +1768°C
  Sensibilidad: ~6 µV/°C
  Aplicaciones: Estándares de calibración, temperaturas muy altas

TIPO B: Platino/Rodio 30%-Platino/Rodio 6%
  Rango: +200°C a +1820°C
  Sensibilidad: ~1 µV/°C (¡muy baja!)
  Aplicaciones: Fabricación de vidrio, hornos muy calientes

Ventajas:
  + Máxima precisión
  + Excelente estabilidad
  + Temperaturas extremas

Desventajas:
  - Costo extremadamente alto
  - Muy baja sensibilidad (ruido)
  - Frágil (requiere protección)


12. CÁLCULOS Y ECUACIONES DE TERMOPARES
--------------------------------------------

12.1 POLINOMIOS NIST ITS-90
----------------------------
El estándar NIST (Instituto Nacional de Estándares y Tecnología) define
polinomios de alta precisión para conversión temperatura ↔ voltaje.

Forma general para TEMPERATURA → VOLTAJE:

  V(T) = c₀ + c₁·T + c₂·T² + c₃·T³ + ... + cₙ·Tⁿ

Donde:
  V = Voltaje en milivoltios (mV)
  T = Temperatura en grados Celsius (°C)
  c₀, c₁, ..., cₙ = Coeficientes polinómicos específicos del tipo

Para Tipo K (0°C a 1372°C):
  V(T) = c₀ + c₁·T + c₂·T² + ... + c₉·T⁹ + a₀·exp[a₁(T-a₂)²]

  Coeficientes (primeros 5):
    c₀ = -0.176004136860 x 10⁻¹
    c₁ =  0.389212049750 x 10⁻¹
    c₂ =  0.185587700320 x 10⁻⁴
    c₃ = -0.994575928740 x 10⁻⁷
    c₄ =  0.318409457190 x 10⁻⁹
    ...más término exponencial


12.2 COMPENSACIÓN DE UNIÓN FRÍA
--------------------------------
Los termopares miden la diferencia de temperatura entre dos uniones.
Para obtener la temperatura absoluta, debe compensarse la temperatura de la
unión fría (Compensación de Unión Fría - CJC).

Temperatura medida = Temperatura calculada de V + Temperatura de unión fría

Métodos CJC:
1. Baño de hielo a 0°C (laboratorio)
2. Sensor RTD/termistor en la unión (industrial)
3. Compensación electrónica (circuitos especializados)

En RTD Master Pro:
- El usuario puede establecer manualmente la temperatura de unión fría
- Valor predeterminado: 25°C (temperatura ambiente)
- Cálculo automático con compensación


12.3 CÁLCULO INVERSO (VOLTAJE → TEMPERATURA)
---------------------------------------------
Para conversión inversa, se utiliza el método de Newton-Raphson para resolver
la ecuación polinómica iterativamente.

Algoritmo:
1. Compensación CJC: V_compensado = V_medido + V(T_unión_fría)
2. Estimación inicial: T₀ = 0°C
3. Iteración de Newton-Raphson:
   T_(n+1) = T_n - [V(T_n) - V_compensado] / [dV/dT(T_n)]
4. Repetir hasta convergencia (|T_(n+1) - T_n| < tolerancia)


13. USO DE TERMOPARES EN RTD MASTER PRO
-------------------------------------------

13.1 SELECCIÓN DEL MODO TERMOPAR
---------------------------------
1. Abrir RTD Master Pro
2. En la sección "Modo de Medición", seleccionar "Termopar"
3. La interfaz cambiará para mostrar controles específicos de termopar

13.2 CONFIGURACIÓN DE TERMOPAR
-------------------------------
1. Tipo: Seleccionar tipo de termopar (K, J, T, E, N, S, R, B)
2. Rango: Se muestra automáticamente para el tipo seleccionado
3. Sensibilidad: Se muestra automáticamente (µV/°C)
4. Temp. Unión Fría: Establecer temperatura de unión fría (predeterminado 25°C)

13.3 CONVERSIONES SOPORTADAS
-----------------------------

MODO 1: TEMPERATURA → VOLTAJE
  Entrada: Temperatura en °C
  Salida: Voltaje en mV
  Ejemplo Tipo K:
    500°C → 20.644 mV

MODO 2: VOLTAJE → TEMPERATURA
  Entrada: Voltaje en mV
  Salida: Temperatura en °C (con compensación CJC)
  Ejemplo Tipo K:
    20.644 mV (CJC=25°C) → ~500°C

13.4 EJEMPLOS PRÁCTICOS
-----------------------

Ejemplo 1: Medición de temperatura de horno con Tipo K
  - Voltaje medido: 35.2 mV
  - Unión fría: 23°C
  - Seleccionar: Tipo K, Voltaje → Temperatura
  - Entrada: 35.2 mV, CJC: 23°C
  - Salida: ~852°C

Ejemplo 2: Calcular voltaje esperado Tipo J
  - Temperatura del horno: 600°C
  - Seleccionar: Tipo J, Temperatura → Voltaje
  - Entrada: 600°C
  - Salida: ~33.277 mV
