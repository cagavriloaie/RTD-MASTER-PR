1. ОБЗОР
================================================================================

RTD Master Pro - это профессиональное настольное приложение, предназначенное для
точных температурных измерений с использованием как датчиков RTD (термометров
сопротивления), так и термопар, в соответствии с множеством международных
стандартов.

ДАТЧИКИ RTD:
Прецизионные устройства измерения температуры, которые используют предсказуемое
изменение электрического сопротивления чистых металлов с температурой.

ТЕРМОПАРЫ:
Температурные датчики, которые генерируют электрическое напряжение на основе
эффекта Зеебека, идеально подходят для экстремальных температур и быстрого
отклика.

Приложение реализует стандартизированные уравнения и коэффициенты, определенные
международными метрологическими организациями (IEC, NIST, ASTM, GOST).

Датчики RTD являются прецизионными устройствами измерения температуры, которые
используют предсказуемое изменение электрического сопротивления чистых металлов
с температурой. Это приложение реализует стандартизированные уравнения и
коэффициенты, определенные международными метрологическими организациями.

2. МЕЖДУНАРОДНЫЕ СТАНДАРТЫ

2.1 IEC 60751:2008
-------------------
Название: "Промышленные платиновые термометры сопротивления и платиновые
          температурные датчики"
Организация: Международная электротехническая комиссия
Область применения: Определяет требования и допуски для промышленных платиновых RTD
Температурный диапазон (платина): -200°C до +850°C
Основной коэффициент: α = 0.00385 Ω/Ω/°C (Pt385)
Классы допусков: AA, A, B (ранее 1/10 DIN, 1/3 DIN, DIN)

Стандартные значения: Pt100, Pt500, Pt1000
Коэффициенты Каллендара-Ван Дюзена для Pt385:
  A = 3.9083 x 10^-3  °C^-1
  B = -5.775 x 10^-7  °C^-2
  C = -4.183 x 10^-12 °C^-4 (только для T < 0°C)


2.2 GOST 6651-2009
-------------------
Название: "Термометры сопротивления, термопреобразователи сопротивления и
          терморезисторы"
Организация: Межгосударственный совет по стандартизации (страны СНГ)
Область применения: Российский/советский стандарт для спецификаций RTD
Температурный диапазон (платина): -200°C до +600°C
Основной коэффициент: α = 0.00391 Ω/Ω/°C (Pt391)

Коэффициенты Каллендара-Ван Дюзена для Pt391:
  A =  3.96847 x 10^-3  °C^-1
  B = -5.847   x 10^-7  °C^-2
  C = -4.35    x 10^-12 °C^-4 (только для T < 0°C)

Примечание: Также определяет медные (Cu426) и никелевые RTD


2.3 ASTM E1137/E1137M-20
-------------------------
Название: "Стандартная спецификация для промышленных платиновых термометров
          сопротивления"
Организация: Американское общество по испытаниям и материалам
Область применения: Североамериканский стандарт для промышленных платиновых RTD
Температурный диапазон: -200°C до +850°C
Коэффициент: α = 0.00385 Ω/Ω/°C (такой же, как IEC 60751)

Примечание: ASTM E1137 соответствует IEC 60751 для международной совместимости
Классы допусков: Grade A, Grade B (эквивалентны классам IEC)

Коэффициент: α = 0.00392 Ω/Ω/°C (Pt392)
Температурный диапазон: -200°C до +850°C

Коэффициенты Каллендара-Ван Дюзена для Pt392:
  A =  3.9848 x 10^-3  °C^-1
  B = -5.870  x 10^-7  °C^-2
  C = -4.40   x 10^-12 °C^-4 (только для T < 0°C)

4. МЕТОДЫ РАСЧЕТА И УРАВНЕНИЯ
================================================================================

4.1 РАСЧЕТ ПЛАТИНОВЫХ RTD - Уравнение Каллендара-Ван Дюзена
------------------------------------------------------------

Уравнение Каллендара-Ван Дюзена является международно признанным методом для
расчета сопротивления платиновых RTD как функции температуры.

Для T ≥ 0°C:
  R(T) = R₀(1 + AT + BT²)

Для T < 0°C:
  R(T) = R₀[1 + AT + BT² + C(T - 100)T³]

Где:
  R(T) = Сопротивление при температуре T (в Ω)
  R₀   = Номинальное сопротивление при 0°C (в Ω)
  T    = Температура (в °C)
  A, B, C = Коэффициенты Каллендара-Ван Дюзена

ЗНАЧЕНИЯ КОЭФФИЦИЕНТОВ:

Pt385 (IEC 60751, ASTM E1137):
  A =  3.9083 × 10⁻³  °C⁻¹
  B = -5.775  × 10⁻⁷  °C⁻²
  C = -4.183  × 10⁻¹² °C⁻⁴

Pt391 (GOST 6651):
  A =  3.96847 × 10⁻³  °C⁻¹
  B = -5.847   × 10⁻⁷  °C⁻²
  C = -4.35    × 10⁻¹² °C⁻⁴

Pt392 (Устаревший США):
  A =  3.9848 × 10⁻³  °C⁻¹
  B = -5.870  × 10⁻⁷  °C⁻²
  C = -4.40   × 10⁻¹² °C⁻⁴


ОБРАТНЫЙ РАСЧЕТ (Сопротивление → Температура):

Для R ≥ R₀ (T ≥ 0°C):
  Решение квадратного уравнения: BT² + AT + (1 - R/R₀) = 0

  T = (-A + √(A² - 4B(1 - R/R₀))) / (2B)

Для R < R₀ (T < 0°C):
  Решение кубического уравнения итеративно с использованием численных методов
  (метод Ньютона-Рафсона)


4.2 РАСЧЕТ МЕДНЫХ И НИКЕЛЕВЫХ RTD - Линейное приближение
---------------------------------------------------------

Медные и никелевые RTD используют упрощенную линейную зависимость из-за их
более узких температурных диапазонов и различных физических свойств.

Прямой расчет (Температура → Сопротивление):
  R(T) = R₀(1 + αT)

Обратный расчет (Сопротивление → Температура):
  T = (R - R₀) / (R₀ × α)

Где:
  R(T) = Сопротивление при температуре T (в Ω)
  R₀   = Номинальное сопротивление при 0°C (в Ω)
  α    = Температурный коэффициент сопротивления (в Ω/Ω/°C)
  T    = Температура (в °C)


МЕДНЫЕ КОЭФФИЦИЕНТЫ:
  α₄₂₆ = 0.00426 Ω/Ω/°C
  α₄₂₇ = 0.00427 Ω/Ω/°C
  α₄₂₈ = 0.00428 Ω/Ω/°C

НИКЕЛЕВЫЕ КОЭФФИЦИЕНТЫ:
  α₆₁₇ = 0.00617 Ω/Ω/°C (Ni100)
  α₆₁₈ = 0.00618 Ω/Ω/°C (Ni1000)
  α₆₇₂ = 0.00672 Ω/Ω/°C (Ni120)

5.1 КЛАССЫ ДОПУСКОВ IEC 60751 (Платиновые RTD)
-----------------------------------------------

Класс AA (Наивысшая точность):
  Допуск = ±(0.1 + 0.0017|t|) °C

  Примеры:
    При   0°C: ±0.10°C
    При  50°C: ±0.185°C
    При 100°C: ±0.27°C
    При 200°C: ±0.44°C

Класс A (Стандартная точность):
  Допуск = ±(0.15 + 0.002|t|) °C

  Примеры:
    При   0°C: ±0.15°C
    При  50°C: ±0.25°C
    При 100°C: ±0.35°C
    При 200°C: ±0.55°C

Класс B (Общего назначения):
  Допуск = ±(0.3 + 0.005|t|) °C

  Примеры:
    При   0°C: ±0.30°C
    При  50°C: ±0.55°C
    При 100°C: ±0.80°C
    При 200°C: ±1.30°C

Класс C (Пониженная точность):
  Допуск = ±(0.6 + 0.01|t|) °C

  Примеры:
    При   0°C: ±0.60°C
    При  50°C: ±1.10°C
    При 100°C: ±1.60°C
    При 200°C: ±2.60°C

Где |t| - абсолютное значение температуры в градусах Цельсия


5.2 ДОПУСК СОПРОТИВЛЕНИЯ ПРИ 0°C
---------------------------------

Для датчиков Pt100:
  Класс AA: ±0.04 Ω при 0°C
  Класс A:  ±0.06 Ω при 0°C
  Класс B:  ±0.12 Ω при 0°C
  Класс C:  ±0.24 Ω при 0°C

  Температура в сопротивление: R = 100(1 + 0.00427T)

Ni100 при температуре T (Ni617):
  0°C:  R = 100.00 Ω
  Температура в сопротивление: R = 100(1 + 0.00617T)

10. ТЕРМОПАРЫ - ОБЩИЙ ОБЗОР
-------------------------------------

10.1 ЧТО ТАКОЕ ТЕРМОПАРЫ?
--------------------------
Термопара - это температурный датчик, который генерирует электрическое напряжение
(термоэлектрическую электродвижущую силу), пропорциональное разности температур
между двумя спаями разнородных металлов. Она основана на эффекте Зеебека,
открытом в 1821 году Томасом Иоганном Зеебеком.

Принцип работы:
- Два провода из разных металлов соединены на одном конце (измерительный спай)
- Другой конец находится при эталонной температуре (холодный спай)
- Разность температур генерирует небольшое напряжение (µV-mV)
- Напряжение прямо пропорционально измеряемой температуре

11. ТИПЫ И СПЕЦИФИКАЦИИ ТЕРМОПАР
-----------------------------------------

11.1 ТИП K (ХРОМЕЛЬ-АЛЮМЕЛЬ)
-----------------------------
Наиболее распространенный тип термопары для общего назначения и промышленного
применения.

Материалы:
  Положительный (+): Хромель (90% Никель, 10% Хром)
  Отрицательный (-): Алюмель (95% Никель, 2% Марганец, 2% Алюминий, 1% Кремний)

Спецификации:
  Температурный диапазон: -270°C до +1372°C
  Чувствительность при 0°C: ~41 µV/°C
  Цветовой код: Желтый
  Стандарт: IEC 60584, ASTM E230

Применения:
  - Промышленность общего назначения
  - Системы HVAC
  - Управление процессами
  - Печи и котлы

Преимущества:
  + Широкий температурный диапазон
  + Низкая стоимость
  + Хорошая стойкость к окислению
  + Широкая доступность

Недостатки:
  - Магнитный дрейф выше 800°C
  - Не подходит для восстановительных атмосфер
  - Умеренная точность


11.2 ТИП J (ЖЕЛЕЗО-КОНСТАНТАН)
-------------------------------
Термопара для железной/сталелитейной промышленности и восстановительных атмосфер.

Материалы:
  Положительный (+): Железо (Fe)
  Отрицательный (-): Константан (55% Медь, 45% Никель)

Спецификации:
  Температурный диапазон: -210°C до +1200°C
  Чувствительность при 0°C: ~52 µV/°C (наивысшая)
  Цветовой код: Черный
  Стандарт: IEC 60584, ASTM E230

Применения:
  - Железная и сталелитейная промышленность
  - Восстановительные атмосферы
  - Бюджетные применения
  - Устаревшее оборудование

Преимущества:
  + Высокая чувствительность
  + Очень низкая стоимость
  + Хорошо для средних температур

Недостатки:
  - Быстрое окисление выше 540°C
  - Не для окислительных атмосфер
  - Более ограниченный диапазон, чем тип K


11.3 ТИП T (МЕДЬ-КОНСТАНТАН)
-----------------------------
Термопара для криогенных и лабораторных применений.

Материалы:
  Положительный (+): Медь (Cu)
  Отрицательный (-): Константан (55% Медь, 45% Никель)

Спецификации:
  Температурный диапазон: -270°C до +400°C
  Чувствительность при 0°C: ~41 µV/°C
  Цветовой код: Синий
  Стандарт: IEC 60584, ASTM E230

Применения:
  - Криогеника и низкие температуры
  - Пищевая промышленность
  - Лаборатория и исследования
  - Коррозионные среды

Преимущества:
  + Превосходна для низких температур
  + Очень хорошая линейность
  + Устойчива к влаге
  + Хорошая точность

Недостатки:
  - Ограниченный диапазон (макс. 400°C)
  - Медь окисляется выше 400°C


11.4 ТИП E (ХРОМЕЛЬ-КОНСТАНТАН)
--------------------------------
Термопара с наивысшей чувствительностью (максимальный выходной сигнал).

Материалы:
  Положительный (+): Хромель (90% Никель, 10% Хром)
  Отрицательный (-): Константан (55% Медь, 45% Никель)

Спецификации:
  Температурный диапазон: -270°C до +1000°C
  Чувствительность при 0°C: ~61 µV/°C (максимальная!)
  Цветовой код: Фиолетовый
  Стандарт: IEC 60584, ASTM E230

Применения:
  - Криогеника
  - Производство энергии
  - Точное обнаружение малых изменений

Преимущества:
  + Максимальная чувствительность (большой сигнал)
  + Превосходное соотношение сигнал/шум
  + Хорошая коррозионная стойкость

Недостатки:
  - Более высокая стоимость
  - Менее распространена (ограниченная доступность)


11.5 ТИП N (НИКРОСИЛ-НИСИЛ)
----------------------------
Современная термопара с улучшенной стабильностью при высоких температурах.

Материалы:
  Положительный (+): Никросил (84.4% Ni, 14.2% Cr, 1.4% Si)
  Отрицательный (-): Нисил (95.5% Ni, 4.4% Si, 0.1% Mg)

Спецификации:
  Температурный диапазон: -270°C до +1300°C
  Чувствительность при 0°C: ~27 µV/°C
  Цветовой код: Оранжевый
  Стандарт: IEC 60584

Применения:
  - Замена типа K (лучшая стабильность)
  - Высокие температуры
  - Окислительные атмосферы

Преимущества:
  + Превосходная стабильность по сравнению с типом K
  + Нет магнитного дрейфа
  + Превосходная стойкость к окислению

Недостатки:
  - Более высокая стоимость, чем тип K
  - Более низкая чувствительность


11.6 ТИПЫ S, R, B (БЛАГОРОДНЫЕ МЕТАЛЛЫ - ПЛАТИНА)
--------------------------------------------------
Платиновые термопары для очень высоких температур и максимальной точности.

ТИП S: Платина-Платина/Родий 10%
  Диапазон: -50°C до +1768°C
  Чувствительность: ~6 µV/°C
  Применения: Фармацевтика, биотехнологии, лабораторные эталоны

ТИП R: Платина-Платина/Родий 13%
  Диапазон: -50°C до +1768°C
  Чувствительность: ~6 µV/°C
  Применения: Калибровочные эталоны, очень высокие температуры

ТИП B: Платина/Родий 30%-Платина/Родий 6%
  Диапазон: +200°C до +1820°C
  Чувствительность: ~1 µV/°C (очень низкая!)
  Применения: Производство стекла, очень горячие печи

Преимущества:
  + Максимальная точность
  + Превосходная стабильность
  + Экстремальные температуры

Недостатки:
  - Чрезвычайно высокая стоимость
  - Очень низкая чувствительность (шум)
  - Хрупкие (требуют защиты)


12. РАСЧЕТЫ И УРАВНЕНИЯ ДЛЯ ТЕРМОПАР
--------------------------------------------

12.1 ПОЛИНОМЫ NIST ITS-90
--------------------------
Стандарт NIST (Национальный институт стандартов и технологий) определяет
высокоточные полиномы для преобразования температура ↔ напряжение.

Общая форма для ТЕМПЕРАТУРА → НАПРЯЖЕНИЕ:

  V(T) = c₀ + c₁·T + c₂·T² + c₃·T³ + ... + cₙ·Tⁿ

Где:
  V = Напряжение в милливольтах (mV)
  T = Температура в градусах Цельсия (°C)
  c₀, c₁, ..., cₙ = Полиномиальные коэффициенты, специфичные для типа

Для типа K (0°C до 1372°C):
  V(T) = c₀ + c₁·T + c₂·T² + ... + c₉·T⁹ + a₀·exp[a₁(T-a₂)²]

  Коэффициенты (первые 5):
    c₀ = -0.176004136860 x 10⁻¹
    c₁ =  0.389212049750 x 10⁻¹
    c₂ =  0.185587700320 x 10⁻⁴
    c₃ = -0.994575928740 x 10⁻⁷
    c₄ =  0.318409457190 x 10⁻⁹
    ...плюс экспоненциальный член


12.2 КОМПЕНСАЦИЯ ХОЛОДНОГО СПАЯ
---------------------------------
Термопары измеряют разность температур между двумя спаями.
Для получения абсолютной температуры необходимо компенсировать температуру
холодного спая (Компенсация Холодного Спая - CJC).

Измеренная температура = Рассчитанная температура из V + Температура холодного спая

Методы CJC:
1. Ледяная ванна при 0°C (лаборатория)
2. RTD/термисторный датчик в спае (промышленность)
3. Электронная компенсация (специализированные схемы)

В RTD Master Pro:
- Пользователь может вручную установить температуру холодного спая
- Значение по умолчанию: 25°C (комнатная температура)
- Автоматический расчет с компенсацией


12.3 ОБРАТНЫЙ РАСЧЕТ (НАПРЯЖЕНИЕ → ТЕМПЕРАТУРА)
------------------------------------------------
Для обратного преобразования используется метод Ньютона-Рафсона для
итеративного решения полиномиального уравнения.

Алгоритм:
1. Компенсация CJC: V_компенсированное = V_измеренное + V(T_холодный_спай)
2. Начальное приближение: T₀ = 0°C
3. Итерация Ньютона-Рафсона:
   T_(n+1) = T_n - [V(T_n) - V_компенсированное] / [dV/dT(T_n)]
4. Повторять до сходимости (|T_(n+1) - T_n| < допуск)


13. ИСПОЛЬЗОВАНИЕ ТЕРМОПАР В RTD MASTER PRO
-------------------------------------------

13.1 ВЫБОР РЕЖИМА ТЕРМОПАРЫ
----------------------------
1. Открыть RTD Master Pro
2. В разделе "Режим измерения" выбрать "Термопара"
3. Интерфейс переключится для отображения элементов управления, специфичных
   для термопар

13.2 КОНФИГУРАЦИЯ ТЕРМОПАРЫ
-----------------------------
1. Тип: Выбрать тип термопары (K, J, T, E, N, S, R, B)
2. Диапазон: Отображается автоматически для выбранного типа
3. Чувствительность: Отображается автоматически (µV/°C)
4. Температура холодного спая: Установить температуру холодного спая
   (по умолчанию 25°C)

13.3 ПОДДЕРЖИВАЕМЫЕ ПРЕОБРАЗОВАНИЯ
------------------------------------

РЕЖИМ 1: ТЕМПЕРАТУРА → НАПРЯЖЕНИЕ
  Вход: Температура в °C
  Выход: Напряжение в mV
  Пример тип K:
    500°C → 20.644 mV

РЕЖИМ 2: НАПРЯЖЕНИЕ → ТЕМПЕРАТУРА
  Вход: Напряжение в mV
  Выход: Температура в °C (с компенсацией CJC)
  Пример тип K:
    20.644 mV (CJC=25°C) → ~500°C

13.4 ПРАКТИЧЕСКИЕ ПРИМЕРЫ
--------------------------

Пример 1: Измерение температуры печи термопарой типа K
  - Измеренное напряжение: 35.2 mV
  - Холодный спай: 23°C
  - Выбрать: Тип K, Напряжение → Температура
  - Ввод: 35.2 mV, CJC: 23°C
  - Выход: ~852°C

Пример 2: Расчет ожидаемого напряжения тип J
  - Температура печи: 600°C
  - Выбрать: Тип J, Температура → Напряжение
  - Ввод: 600°C
  - Выход: ~33.277 mV
