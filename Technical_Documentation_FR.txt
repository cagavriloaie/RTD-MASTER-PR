1. APERÇU GÉNÉRAL
================================================================================

RTD Master Pro est une application de bureau de qualité professionnelle conçue
pour des mesures de température précises utilisant à la fois les capteurs RTD
(Détecteurs de Température à Résistance) et les thermocouples, conformément à
plusieurs normes internationales.

CAPTEURS RTD :
Dispositifs de mesure de température de précision qui exploitent le changement
prévisible de la résistance électrique des métaux purs en fonction de la
température.

THERMOCOUPLES :
Capteurs de température qui génèrent une tension électrique basée sur l'effet
Seebeck, idéaux pour les températures extrêmes et la réponse rapide.

L'application implémente les équations et coefficients standardisés définis par
les organisations internationales de métrologie (IEC, NIST, ASTM, GOST).

Les capteurs RTD sont des dispositifs de mesure de température de précision qui
exploitent le changement prévisible de la résistance électrique des métaux purs
en fonction de la température. Cette application implémente les équations et
coefficients standardisés définis par les organisations internationales de
métrologie.

4. MÉTHODES DE CALCUL ET ÉQUATIONS
================================================================================

4.1 CALCUL RTD AU PLATINE - Équation de Callendar-Van Dusen
------------------------------------------------------------

L'équation de Callendar-Van Dusen est la méthode internationalement acceptée
pour calculer la résistance RTD au platine en fonction de la température.

Pour T ≥ 0°C :
  R(T) = R₀(1 + AT + BT²)

Pour T < 0°C :
  R(T) = R₀[1 + AT + BT² + C(T - 100)T³]

Où :
  R(T) = Résistance à la température T (en Ω)
  R₀   = Résistance nominale à 0°C (en Ω)
  T    = Température (en °C)
  A, B, C = Coefficients de Callendar-Van Dusen

VALEURS DES COEFFICIENTS :

Pt385 (IEC 60751, ASTM E1137) :
  A =  3,9083 × 10⁻³  °C⁻¹
  B = -5,775  × 10⁻⁷  °C⁻²
  C = -4,183  × 10⁻¹² °C⁻⁴

Pt391 (GOST 6651) :
  A =  3,96847 × 10⁻³  °C⁻¹
  B = -5,847   × 10⁻⁷  °C⁻²
  C = -4,35    × 10⁻¹² °C⁻⁴

Pt392 (Héritage américain) :
  A =  3,9848 × 10⁻³  °C⁻¹
  B = -5,870  × 10⁻⁷  °C⁻²
  C = -4,40   × 10⁻¹² °C⁻⁴


CALCUL INVERSE (Résistance → Température) :

Pour R ≥ R₀ (T ≥ 0°C) :
  Résoudre l'équation quadratique : BT² + AT + (1 - R/R₀) = 0

  T = (-A + √(A² - 4B(1 - R/R₀))) / (2B)

Pour R < R₀ (T < 0°C) :
  Résoudre l'équation cubique de manière itérative en utilisant des méthodes
  numériques (Newton-Raphson)


4.2 CALCUL RTD CUIVRE ET NICKEL - Approximation linéaire
---------------------------------------------------------

Les RTD en cuivre et en nickel utilisent une relation linéaire simplifiée en
raison de leurs plages de température plus étroites et de leurs propriétés
physiques différentes.

Calcul direct (Température → Résistance) :
  R(T) = R₀(1 + αT)

Calcul inverse (Résistance → Température) :
  T = (R - R₀) / (R₀ × α)

Où :
  R(T) = Résistance à la température T (en Ω)
  R₀   = Résistance nominale à 0°C (en Ω)
  α    = Coefficient de température de résistance (en Ω/Ω/°C)
  T    = Température (en °C)


COEFFICIENTS CUIVRE :
  α₄₂₆ = 0,00426 Ω/Ω/°C
  α₄₂₇ = 0,00427 Ω/Ω/°C
  α₄₂₈ = 0,00428 Ω/Ω/°C

COEFFICIENTS NICKEL :
  α₆₁₇ = 0,00617 Ω/Ω/°C (Ni100)
  α₆₁₈ = 0,00618 Ω/Ω/°C (Ni1000)
  α₆₇₂ = 0,00672 Ω/Ω/°C (Ni120)

5. SPÉCIFICATIONS DE TOLÉRANCE
================================================================================

5.1 CLASSES DE TOLÉRANCE IEC 60751 (RTD au platine)
----------------------------------------------------

Classe AA (Précision la plus élevée) :
  Tolérance = ±(0,1 + 0,0017|t|) °C

  Exemples :
    À   0°C : ±0,10°C
    À  50°C : ±0,185°C
    À 100°C : ±0,27°C
    À 200°C : ±0,44°C

Classe A (Précision standard) :
  Tolérance = ±(0,15 + 0,002|t|) °C

  Exemples :
    À   0°C : ±0,15°C
    À  50°C : ±0,25°C
    À 100°C : ±0,35°C
    À 200°C : ±0,55°C

Classe B (Usage général) :
  Tolérance = ±(0,3 + 0,005|t|) °C

  Exemples :
    À   0°C : ±0,30°C
    À  50°C : ±0,55°C
    À 100°C : ±0,80°C
    À 200°C : ±1,30°C

Classe C (Précision inférieure) :
  Tolérance = ±(0,6 + 0,01|t|) °C

  Exemples :
    À   0°C : ±0,60°C
    À  50°C : ±1,10°C
    À 100°C : ±1,60°C
    À 200°C : ±2,60°C

Où |t| est la valeur absolue de la température en °C


5.2 TOLÉRANCE DE RÉSISTANCE À 0°C
----------------------------------

Pour les capteurs Pt100 :
  Classe AA : ±0,04 Ω à 0°C
  Classe A :  ±0,06 Ω à 0°C
  Classe B :  ±0,12 Ω à 0°C
  Classe C :  ±0,24 Ω à 0°C

10.1 QU'EST-CE QU'UN THERMOCOUPLE ?
------------------------------------
Un thermocouple est un capteur de température qui génère une tension électrique
(force électromotrice thermoélectrique) proportionnelle à la différence de
température entre deux jonctions de métaux différents. Il est basé sur l'effet
Seebeck, découvert en 1821 par Thomas Johann Seebeck.

Principe de fonctionnement :
- Deux fils de métaux différents sont joints à une extrémité (jonction de mesure)
- L'autre extrémité est à une température de référence (jonction froide)
- La différence de température génère une petite tension (µV-mV)
- La tension est directement proportionnelle à la température mesurée

-------------------------------------

11. TYPES ET SPÉCIFICATIONS DES THERMOCOUPLES
-----------------------------------------

11.1 TYPE K (CHROMEL-ALUMEL)
----------------------------
Le type de thermocouple le plus courant pour usage général et industriel.

Matériaux :
  Positif (+) : Chromel (90% Nickel, 10% Chrome)
  Négatif (-) : Alumel (95% Nickel, 2% Manganèse, 2% Aluminium, 1% Silicium)

Spécifications :
  Plage de température : -270°C à +1372°C
  Sensibilité à 0°C : ~41 µV/°C
  Code couleur : Jaune
  Norme : IEC 60584, ASTM E230

Applications :
  - Industriel usage général
  - Systèmes CVC
  - Contrôle de processus
  - Fours et chaudières

Avantages :
  + Large plage de température
  + Faible coût
  + Bonne résistance à l'oxydation
  + Largement disponible

Inconvénients :
  - Dérive magnétique au-dessus de 800°C
  - Non adapté aux atmosphères réductrices
  - Précision modérée


11.2 TYPE J (FER-CONSTANTAN)
-----------------------------
Thermocouple pour l'industrie du fer/acier et atmosphères réductrices.

Matériaux :
  Positif (+) : Fer (Fe)
  Négatif (-) : Constantan (55% Cuivre, 45% Nickel)

Spécifications :
  Plage de température : -210°C à +1200°C
  Sensibilité à 0°C : ~52 µV/°C (la plus élevée)
  Code couleur : Noir
  Norme : IEC 60584, ASTM E230

Applications :
  - Industrie du fer et de l'acier
  - Atmosphères réductrices
  - Applications à budget limité
  - Équipement hérité

Avantages :
  + Haute sensibilité
  + Très faible coût
  + Bon pour températures moyennes

Inconvénients :
  - Oxydation rapide au-dessus de 540°C
  - Non adapté aux atmosphères oxydantes
  - Plage plus limitée que le Type K


11.3 TYPE T (CUIVRE-CONSTANTAN)
--------------------------------

  Sensibilité à 0°C : ~27 µV/°C
  Code couleur : Orange

12. CALCULS ET ÉQUATIONS DES THERMOCOUPLES
--------------------------------------------

12.1 POLYNÔMES NIST ITS-90
---------------------------
La norme NIST (National Institute of Standards and Technology) définit des
polynômes de haute précision pour la conversion température ↔ tension.

Forme générale pour TEMPÉRATURE → TENSION :

  V(T) = c₀ + c₁·T + c₂·T² + c₃·T³ + ... + cₙ·Tⁿ

Où :
  V = Tension en millivolts (mV)
  T = Température en degrés Celsius (°C)
  c₀, c₁, ..., cₙ = Coefficients polynomiaux spécifiques au type

Pour Type K (0°C à 1372°C) :
  V(T) = c₀ + c₁·T + c₂·T² + ... + c₉·T⁹ + a₀·exp[a₁(T-a₂)²]

  Coefficients (5 premiers) :
    c₀ = -0,176004136860 x 10⁻¹
    c₁ =  0,389212049750 x 10⁻¹
    c₂ =  0,185587700320 x 10⁻⁴
    c₃ = -0,994575928740 x 10⁻⁷
    c₄ =  0,318409457190 x 10⁻⁹
    ...plus terme exponentiel


12.2 COMPENSATION DE JONCTION FROIDE
-------------------------------------
Les thermocouples mesurent la différence de température entre deux jonctions.
Pour obtenir la température absolue, la température de la jonction froide doit
être compensée (Compensation de Jonction Froide - CJC).

Température mesurée = Température calculée à partir de V + Température jonction froide

Méthodes CJC :
1. Bain de glace à 0°C (laboratoire)
2. Capteur RTD/thermistance à la jonction (industriel)
3. Compensation électronique (circuits spécialisés)

Dans RTD Master Pro :
- L'utilisateur peut définir manuellement la température de jonction froide
- Valeur par défaut : 25°C (température ambiante)
- Calcul automatique avec compensation


12.3 CALCUL INVERSE (TENSION → TEMPÉRATURE)
--------------------------------------------
Pour la conversion inverse, la méthode de Newton-Raphson est utilisée pour
résoudre l'équation polynomiale de manière itérative.

Algorithme :
1. Compensation CJC : V_compensée = V_mesurée + V(T_jonction_froide)
2. Estimation initiale : T₀ = 0°C
3. Itération Newton-Raphson :
   T_(n+1) = T_n - [V(T_n) - V_compensée] / [dV/dT(T_n)]
4. Répéter jusqu'à convergence (|T_(n+1) - T_n| < tolérance)
