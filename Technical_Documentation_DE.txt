1. ÜBERBLICK
================================================================================

RTD Master Pro ist eine professionelle Desktop-Anwendung für präzise
Temperaturmessungen mit RTD-Sensoren (Widerstandsthermometer) und
Thermoelementen gemäß mehreren internationalen Standards.

RTD-SENSOREN:
Präzise Temperaturmessgeräte, die die vorhersagbare Änderung des
elektrischen Widerstands reiner Metalle mit der Temperatur ausnutzen.

THERMOELEMENTE:
Temperatursensoren, die elektrische Spannung basierend auf dem Seebeck-Effekt
erzeugen, ideal für extreme Temperaturen und schnelle Ansprechzeiten.

Die Anwendung implementiert standardisierte Gleichungen und Koeffizienten,
die von internationalen Metrologie-Organisationen definiert wurden
(IEC, NIST, ASTM, GOST).

RTD-Sensoren sind Präzisionstemperaturmessgeräte, die die vorhersehbare
Änderung des elektrischen Widerstands reiner Metalle mit der Temperatur nutzen.
Diese Anwendung implementiert die standardisierten Gleichungen und Koeffizienten,
die von internationalen metrologischen Organisationen definiert wurden.

4. BERECHNUNGSMETHODEN UND GLEICHUNGEN
================================================================================

4.1 PLATIN-RTD-BERECHNUNG - Callendar-Van Dusen Gleichung
-----------------------------------------------------------

Die Callendar-Van Dusen-Gleichung ist die international anerkannte Methode zur
Berechnung des Platin-RTD-Widerstands als Funktion der Temperatur.

Für T >= 0°C:
  R(T) = R0(1 + AT + BT^2)

Für T < 0°C:
  R(T) = R0[1 + AT + BT^2 + C(T - 100)T^3]

Wobei:
  R(T) = Widerstand bei Temperatur T (in Ohm)
  R0   = Nennwiderstand bei 0°C (in Ohm)
  T    = Temperatur (in °C)
  A, B, C = Callendar-Van Dusen Koeffizienten

KOEFFIZIENTENWERTE:

Pt385 (IEC 60751, ASTM E1137):
  A =  3.9083  x 10^-3  °C^-1
  B = -5.775   x 10^-7  °C^-2
  C = -4.183   x 10^-12 °C^-4

Pt391 (GOST 6651):
  A =  3.96847 x 10^-3  °C^-1
  B = -5.847   x 10^-7  °C^-2
  C = -4.35    x 10^-12 °C^-4

Pt392 (Legacy US):
  A =  3.9848  x 10^-3  °C^-1
  B = -5.870   x 10^-7  °C^-2
  C = -4.40    x 10^-12 °C^-4


UMKEHRBERECHNUNG (Widerstand -> Temperatur):

Für R >= R0 (T >= 0°C):
  Löse quadratische Gleichung: BT^2 + AT + (1 - R/R0) = 0

  T = (-A + sqrt(A^2 - 4B(1 - R/R0))) / (2B)

Für R < R0 (T < 0°C):
  Löse kubische Gleichung iterativ mit numerischen Methoden (Newton-Raphson)


4.2 KUPFER- UND NICKEL-RTD-BERECHNUNG - Lineare Näherung
----------------------------------------------------------

Kupfer- und Nickel-RTDs verwenden eine vereinfachte lineare Beziehung aufgrund
ihrer engeren Temperaturbereiche und unterschiedlichen physikalischen
Eigenschaften.

Vorwärtsberechnung (Temperatur -> Widerstand):
  R(T) = R0(1 + alpha*T)

Umkehrberechnung (Widerstand -> Temperatur):
  T = (R - R0) / (R0 x alpha)

Wobei:
  R(T) = Widerstand bei Temperatur T (in Ohm)
  R0   = Nennwiderstand bei 0°C (in Ohm)
  alpha = Temperaturkoeffizient des Widerstands (in Ohm/Ohm/°C)
  T    = Temperatur (in °C)


KUPFERKOEFFIZIENTEN:
  alpha_426 = 0.00426 Ohm/Ohm/°C
  alpha_427 = 0.00427 Ohm/Ohm/°C
  alpha_428 = 0.00428 Ohm/Ohm/°C

NICKELKOEFFIZIENTEN:
  alpha_617 = 0.00617 Ohm/Ohm/°C (Ni100)
  alpha_618 = 0.00618 Ohm/Ohm/°C (Ni1000)
  alpha_672 = 0.00672 Ohm/Ohm/°C (Ni120)

---------------------------------------------

Klasse AA (Höchste Präzision):
  Toleranz = ±(0.1 + 0.0017|t|) °C

  Beispiele:
    Bei   0°C: ±0.10°C
    Bei  50°C: ±0.185°C
    Bei 100°C: ±0.27°C
    Bei 200°C: ±0.44°C

Klasse A (Standardpräzision):
  Toleranz = ±(0.15 + 0.002|t|) °C

  Beispiele:
    Bei   0°C: ±0.15°C
    Bei  50°C: ±0.25°C
    Bei 100°C: ±0.35°C
    Bei 200°C: ±0.55°C

Klasse B (Allgemeine Verwendung):
  Toleranz = ±(0.3 + 0.005|t|) °C

  Beispiele:
    Bei   0°C: ±0.30°C
    Bei  50°C: ±0.55°C
    Bei 100°C: ±0.80°C
    Bei 200°C: ±1.30°C

Klasse C (Niedrigere Präzision):
  Toleranz = ±(0.6 + 0.01|t|) °C

  Beispiele:
    Bei   0°C: ±0.60°C
    Bei  50°C: ±1.10°C
    Bei 100°C: ±1.60°C

Wobei |t| der Absolutwert der Temperatur in °C ist

10. THERMOELEMENTE - ALLGEMEINE ÜBERSICHT
-------------------------------------

10.1 WAS SIND THERMOELEMENTE?
-------------------------------
Ein Thermoelement ist ein Temperatursensor, der eine elektrische Spannung
(thermoelektrische elektromotorische Kraft) erzeugt, die proportional zur
Temperaturdifferenz zwischen zwei Verbindungsstellen verschiedener Metalle ist.
Es basiert auf dem Seebeck-Effekt, entdeckt 1821 von Thomas Johann Seebeck.

Funktionsprinzip:
- Zwei Drähte aus unterschiedlichen Metallen sind an einem Ende verbunden (Messstelle)
- Das andere Ende befindet sich bei einer Referenztemperatur (Vergleichsstelle)
- Die Temperaturdifferenz erzeugt eine kleine Spannung (µV-mV)
- Die Spannung ist direkt proportional zur gemessenen Temperatur

-------------------------------------

11. THERMOELEMENT-TYPEN UND SPEZIFIKATIONEN
-----------------------------------------

11.1 TYP K (CHROMEL-ALUMEL)
----------------------------
Der gebräuchlichste Thermoelement-Typ für allgemeine und industrielle Anwendungen.

Materialien:
  Positiv (+): Chromel (90% Nickel, 10% Chrom)
  Negativ (-): Alumel (95% Nickel, 2% Mangan, 2% Aluminium, 1% Silizium)

Spezifikationen:
  Temperaturbereich: -270°C bis +1372°C
  Empfindlichkeit bei 0°C: ~41 µV/°C
  Farbcode: Gelb
  Standard: IEC 60584, ASTM E230

Anwendungen:
  - Allgemeine industrielle Zwecke
  - HVAC-Systeme
  - Prozesssteuerung
  - Öfen und Kessel

Vorteile:
  + Breiter Temperaturbereich
  + Niedrige Kosten
  + Gute Oxidationsbeständigkeit
  + Weit verbreitet

Nachteile:
  - Magnetische Drift über 800°C
  - Nicht geeignet für reduzierende Atmosphären
  - Mäßige Genauigkeit


11.2 TYP J (EISEN-KONSTANTAN)
------------------------------
Thermoelement für Eisen-/Stahlindustrie und reduzierende Atmosphären.

Materialien:
  Positiv (+): Eisen (Fe)
  Negativ (-): Konstantan (55% Kupfer, 45% Nickel)

Spezifikationen:
  Temperaturbereich: -210°C bis +1200°C
  Empfindlichkeit bei 0°C: ~52 µV/°C (höchste)
  Farbcode: Schwarz
  Standard: IEC 60584, ASTM E230

Anwendungen:
  - Eisen- und Stahlindustrie
  - Reduzierende Atmosphären
  - Budgetbeschränkte Anwendungen
  - Altgeräte

Vorteile:
  + Hohe Empfindlichkeit
  + Sehr niedrige Kosten
  + Gut für mittlere Temperaturen

Nachteile:
  - Schnelle Oxidation über 540°C
  - Nicht für oxidierende Atmosphären
  - Begrenzter Bereich als Typ K


11.3 TYP T (KUPFER-KONSTANTAN)
-------------------------------
Thermoelement für Kryotechnik und Laboranwendungen.

Materialien:
  Positiv (+): Kupfer (Cu)
  Negativ (-): Konstantan (55% Kupfer, 45% Nickel)

Spezifikationen:
  Temperaturbereich: -270°C bis +400°C
  Empfindlichkeit bei 0°C: ~41 µV/°C
  Farbcode: Blau
  Standard: IEC 60584, ASTM E230

Anwendungen:
  - Kryotechnik und niedrige Temperaturen
  - Lebensmittelindustrie
  - Labor und Forschung
  - Korrosive Umgebungen

Vorteile:
  + Ausgezeichnet für niedrige Temperaturen
  + Sehr gute Linearität
  + Feuchtigkeitsbeständig
  + Gute Genauigkeit

Nachteile:
  - Begrenzter Bereich (max 400°C)
  - Kupfer oxidiert über 400°C


11.4 TYP E (CHROMEL-KONSTANTAN)
--------------------------------
Thermoelement mit der höchsten Empfindlichkeit (maximale Ausgabe).

Materialien:
  Positiv (+): Chromel (90% Nickel, 10% Chrom)
  Negativ (-): Konstantan (55% Kupfer, 45% Nickel)

Spezifikationen:
  Temperaturbereich: -270°C bis +1000°C
  Empfindlichkeit bei 0°C: ~61 µV/°C (maximal!)
  Farbcode: Violett
  Standard: IEC 60584, ASTM E230

Anwendungen:
  - Kryotechnik
  - Energieerzeugung
  - Präzise Erkennung kleiner Änderungen

Vorteile:
  + Maximale Empfindlichkeit (großes Signal)
  + Ausgezeichnetes Signal-Rausch-Verhältnis
  + Gute Korrosionsbeständigkeit

Nachteile:

  - Weniger häufig verwendet (begrenzte Verfügbarkeit)


11.5 TYP N (NICROSIL-NISIL)
----------------------------
Modernes Thermoelement mit verbesserter Stabilität bei hohen Temperaturen.

Materialien:
  Positiv (+): Nicrosil (84.4% Ni, 14.2% Cr, 1.4% Si)
  Negativ (-): Nisil (95.5% Ni, 4.4% Si, 0.1% Mg)

Spezifikationen:
  Temperaturbereich: -270°C bis +1300°C
  Empfindlichkeit bei 0°C: ~27 µV/°C
  Farbcode: Orange
  Standard: IEC 60584

Anwendungen:
  - Typ K Ersatz (bessere Stabilität)
  - Hohe Temperaturen
  - Oxidierende Atmosphären

Vorteile:
  + Überlegene Stabilität gegenüber Typ K
  + Keine magnetische Drift
  + Ausgezeichnete Oxidationsbeständigkeit

Nachteile:
  - Höhere Kosten als Typ K
  - Geringere Empfindlichkeit

12. THERMOELEMENT-BERECHNUNGEN UND GLEICHUNGEN
--------------------------------------------

12.1 NIST ITS-90 POLYNOME
--------------------------
Der NIST-Standard (National Institute of Standards and Technology) definiert
hochpräzise Polynome für Temperatur ↔ Spannungsumwandlung.

Allgemeine Form für TEMPERATUR → SPANNUNG:

  V(T) = c₀ + c₁·T + c₂·T² + c₃·T³ + ... + cₙ·Tⁿ

Wobei:
  V = Spannung in Millivolt (mV)
  T = Temperatur in Grad Celsius (°C)
  c₀, c₁, ..., cₙ = Typspezifische Polynomkoeffizienten

Für Typ K (0°C bis 1372°C):
  V(T) = c₀ + c₁·T + c₂·T² + ... + c₉·T⁹ + a₀·exp[a₁(T-a₂)²]

  Koeffizienten (erste 5):
    c₀ = -0.176004136860 x 10⁻¹
    c₁ =  0.389212049750 x 10⁻¹
    c₂ =  0.185587700320 x 10⁻⁴
    c₃ = -0.994575928740 x 10⁻⁷
    c₄ =  0.318409457190 x 10⁻⁹
    ...plus Exponentialterm


12.2 VERGLEICHSSTELLENKOMPENSATION
------------------------------------
Thermoelemente messen die Temperaturdifferenz zwischen zwei Verbindungsstellen.
Um die absolute Temperatur zu erhalten, muss die Vergleichsstellentemperatur
kompensiert werden (Cold Junction Compensation - CJC).

Gemessene Temperatur = Berechnete Temperatur aus V + Vergleichsstellentemperatur

CJC-Methoden:
1. Eisbad bei 0°C (Labor)
2. RTD/Thermistor-Sensor an Verbindungsstelle (industriell)
3. Elektronische Kompensation (spezialisierte Schaltungen)

In RTD Master Pro:
- Benutzer kann Vergleichsstellentemperatur manuell einstellen
- Standardwert: 25°C (Raumtemperatur)
- Automatische Berechnung mit Kompensation


12.3 UMKEHRBERECHNUNG (SPANNUNG → TEMPERATUR)
----------------------------------------------
Für die Umkehrumwandlung wird die Newton-Raphson-Methode verwendet,
um die Polynomgleichung iterativ zu lösen.

Algorithmus:
1. CJC-Kompensation: V_kompensiert = V_gemessen + V(T_Vergleichsstelle)
2. Anfangsschätzung: T₀ = 0°C
3. Newton-Raphson-Iteration:
   T_(n+1) = T_n - [V(T_n) - V_kompensiert] / [dV/dT(T_n)]
4. Wiederholen bis Konvergenz (|T_(n+1) - T_n| < Toleranz)
