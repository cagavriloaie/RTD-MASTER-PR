1. PREZENTARE GENERALĂ
================================================================================

RTD Master Pro este o aplicație desktop profesională proiectată pentru conversii
precise de temperatură folosind atât senzori RTD (Resistance Temperature
Detector) cât și termocuple, conform mai multor standarde internaționale.

SENZORI RTD:
Dispozitive de măsurare precisă a temperaturii care exploatează schimbarea
previzibilă a rezistenței electrice a metalelor pure cu temperatura.

TERMOCUPLE:
Senzori de temperatură care generează tensiune electrică bazată pe efectul
Seebeck, ideali pentru temperaturi extreme și răspuns rapid.

Aplicația implementează ecuațiile și coeficienții standardizați definiți de
organizațiile internaționale de metrologie (IEC, NIST, ASTM, GOST).

4. METODE DE CALCUL ȘI ECUAȚII
================================================================================

4.1 CALCUL RTD PLATINĂ - Ecuația Callendar-Van Dusen
------------------------------------------------------

Ecuația Callendar-Van Dusen este metoda acceptată internațional pentru
calcularea rezistenței RTD din platină în funcție de temperatură.

Pentru T >= 0°C:
  R(T) = R0(1 + AT + BT^2)

Pentru T < 0°C:
  R(T) = R0[1 + AT + BT^2 + C(T - 100)T^3]

Unde:
  R(T) = Rezistența la temperatura T (în Ohm)
  R0   = Rezistența nominală la 0°C (în Ohm)
  T    = Temperatura (în °C)
  A, B, C = Coeficienții Callendar-Van Dusen

VALORI COEFICIENȚI:

Pt385 (IEC 60751, ASTM E1137):
  A =  3.9083  x 10^-3  °C^-1
  B = -5.775   x 10^-7  °C^-2
  C = -4.183   x 10^-12 °C^-4

Pt391 (GOST 6651):
  A =  3.96847 x 10^-3  °C^-1
  B = -5.847   x 10^-7  °C^-2
  C = -4.35    x 10^-12 °C^-4

Pt392 (Legacy US):
  A =  3.9848  x 10^-3  °C^-1
  B = -5.870   x 10^-7  °C^-2
  C = -4.40    x 10^-12 °C^-4


CALCUL INVERS (Rezistență -> Temperatură):

Pentru R >= R0 (T >= 0°C):
  Rezolvă ecuația de gradul doi: BT^2 + AT + (1 - R/R0) = 0

  T = (-A + sqrt(A^2 - 4B(1 - R/R0))) / (2B)

Pentru R < R0 (T < 0°C):
  Rezolvă ecuația cubică iterativ folosind metode numerice (Newton-Raphson)


4.2 CALCUL RTD CUPRU ȘI NICHEL - Aproximare Liniară
-----------------------------------------------------

RTD-urile din cupru și nichel folosesc o relație liniară simplificată datorită
domeniilor lor mai înguste de temperatură și proprietăților fizice diferite.

Calcul Direct (Temperatură -> Rezistență):
  R(T) = R0(1 + alpha*T)

Calcul Invers (Rezistență -> Temperatură):
  T = (R - R0) / (R0 x alpha)

Unde:
  R(T) = Rezistența la temperatura T (în Ohm)
  R0   = Rezistența nominală la 0°C (în Ohm)
  alpha = Coeficientul de temperatură al rezistenței (în Ohm/Ohm/°C)
  T    = Temperatura (în °C)


COEFICIENȚI CUPRU:
  alpha_426 = 0.00426 Ohm/Ohm/°C
  alpha_427 = 0.00427 Ohm/Ohm/°C
  alpha_428 = 0.00428 Ohm/Ohm/°C

COEFICIENȚI NICHEL:
  alpha_617 = 0.00617 Ohm/Ohm/°C (Ni100)
  alpha_618 = 0.00618 Ohm/Ohm/°C (Ni1000)
  alpha_672 = 0.00672 Ohm/Ohm/°C (Ni120)

5. SPECIFICAȚII DE TOLERANȚĂ
================================================================================

5.1 CLASE DE TOLERANȚĂ IEC 60751 (RTD-uri din Platină)
--------------------------------------------------------

Clasa AA (Precizie Maximă):
  Toleranță = ±(0.1 + 0.0017|t|) °C

  Exemple:
    La   0°C: ±0.10°C
    La  50°C: ±0.185°C
    La 100°C: ±0.27°C
    La 200°C: ±0.44°C

Clasa A (Precizie Standard):
  Toleranță = ±(0.15 + 0.002|t|) °C

  Exemple:
    La   0°C: ±0.15°C
    La  50°C: ±0.25°C
    La 100°C: ±0.35°C
    La 200°C: ±0.55°C

Clasa B (Uz General):
  Toleranță = ±(0.3 + 0.005|t|) °C

  Exemple:
    La   0°C: ±0.30°C
    La  50°C: ±0.55°C
    La 100°C: ±0.80°C
    La 200°C: ±1.30°C

Clasa C (Precizie Redusă):
  Toleranță = ±(0.6 + 0.01|t|) °C

  Exemple:
    La   0°C: ±0.60°C
    La  50°C: ±1.10°C
    La 100°C: ±1.60°C
    La 200°C: ±2.60°C

Unde |t| este valoarea absolută a temperaturii în °C


5.2 TOLERANȚĂ REZISTENȚĂ LA 0°C
---------------------------------

Pentru senzori Pt100:
  Clasa AA: ±0.04 Ohm la 0°C
  Clasa A:  ±0.06 Ohm la 0°C
  Clasa B:  ±0.12 Ohm la 0°C
  Clasa C:  ±0.24 Ohm la 0°C

10. TERMOCUPLE - PREZENTARE GENERALĂ
------------------------------------

10.1 CE SUNT TERMOCUPLELE?
---------------------------
Termocupla este un senzor de temperatură care generează o tensiune electrică
(forță electromotoare termoelectrică) proporțională cu diferența de temperatură
între două joncțiuni de metale diferite. Se bazează pe efectul Seebeck,
descoperit în 1821 de Thomas Johann Seebeck.

Principiu de funcționare:
- Două fire din metale diferite sunt unite la un capăt (joncțiunea măsurare)
- Celălalt capăt este la o temperatură de referință (joncțiunea rece)
- Diferența de temperatură generează o tensiune mică (µV-mV)
- Tensiunea este direct proportională cu temperatura măsurată


10.2 COMPARAȚIE RTD vs TERMOCUPLA
-----------------------------------

┌────────────────────────┬─────────────────────────┬─────────────────────────┐
│ Caracteristică         │ RTD                     │ Termocupla              │
├────────────────────────┼─────────────────────────┼─────────────────────────┤
│ Principiu              │ Rezistență electrică    │ Tensiune termoelectrică │
│ Semnal output          │ Ohm (Ω)                 │ Milivolt (mV)           │
│ Domeniu temperatură    │ -200°C to +850°C        │ -270°C to +1820°C       │
│ Precizie               │ ±0.1°C to ±0.001°C      │ ±0.5°C to ±2°C          │
│ Răspuns               │ Lent (secunde)          │ Rapid (milisecunde)     │
│ Stabilitate            │ Excelentă               │ Bună (deriva în timp)   │
│ Cost                   │ Mai mare                │ Mai mic                 │
│ Alimentare             │ Necesară (curent)       │ Auto-generare           │
│ Dimensiune             │ Mai mare                │ Foarte mic (punct)      │
│ Linearitate            │ Bună                    │ Non-liniară             │
└────────────────────────┴─────────────────────────┴─────────────────────────┘

11.1 TYPE K (CHROMEL-ALUMEL)
-----------------------------
Cel mai comun tip de termocupla pentru uz general și industrial.

Materiale:
  Pozitiv (+): Chromel (90% Nichel, 10% Crom)
  Negativ (-): Alumel (95% Nichel, 2% Mangan, 2% Aluminiu, 1% Silicon)

Specificații:
  Domeniu de temperatură: -270°C to +1372°C
  Sensibilitate la 0°C: ~41 µV/°C
  Cod culoare: Galben (Yellow)
  Standard: IEC 60584, ASTM E230

Aplicații:
  - Uz general industrial
  - Sisteme HVAC
  - Controlul proceselor
  - Cuptoare și cazane

Avantaje:
  + Domeniu larg de temperatură
  + Cost redus
  + Bună rezistență la oxidare
  + Larg disponibil

Dezavantaje:
  - Deriva magnetică peste 800°C
  - Nu este potrivit pentru atmosfere reducătoare
  - Precizie moderată

-------------------------------
Termocupla pentru industria fierului/oțelului și atmosfere reducătoare.

  Pozitiv (+): Fier (Fe)

Specificații:
  Domeniu de temperatură: -210°C to +1200°C
  Sensibilitate la 0°C: ~52 µV/°C (cea mai mare)
  Cod culoare: Negru (Black)
  Standard: IEC 60584, ASTM E230

Aplicații:
  - Industria fierului și oțelului
  - Atmosfere reducătoare
  - Aplicații cu buget limitat
  - Echipamente vechi (legacy)

Avantaje:
  + Sensibilitate ridicată
  + Cost foarte redus
  + Bun pentru temperaturi medii

Dezavantaje:
  - Oxidare rapidă peste 540°C
  - Nu pentru atmosfere oxidante
  - Domeniu mai limitat decât Type K

  - Sensibilitate mai mică


11.6 TYPE S, R, B (METALE NOBLE - PLATINĂ)
--------------------------------------------
Termocuple din platină pentru temperaturi foarte înalte și precizie maximă.

TYPE S: Platină-Platină/Rodiu 10%
  Domeniu: -50°C to +1768°C
  Sensibilitate: ~6 µV/°C
  Aplicații: Farmaceutic, biotech, standarde laborator

TYPE R: Platină-Platină/Rodiu 13%
  Domeniu: -50°C to +1768°C
  Sensibilitate: ~6 µV/°C
  Aplicații: Standarde calibrare, temperaturi foarte înalte

TYPE B: Platină/Rodiu 30%-Platină/Rodiu 6%
  Domeniu: +200°C to +1820°C
  Sensibilitate: ~1 µV/°C (foarte mică!)
  Aplicații: Fabricare sticlă, cuptoare foarte fierbinți

Avantaje:
  + Precizie maximă
  + Stabilitate excelentă
  + Temperaturi extreme

Dezavantaje:
  - Cost extrem de ridicat
  - Sensibilitate foarte mică (zgomot)
  - Fragile (necesită protecție)


12. CALCULE TERMOCUPLE ȘI ECUAȚII
----------------------------------

12.1 POLINOAME NIST ITS-90
---------------------------
Standardul NIST (National Institute of Standards and Technology) definește
polinoame de înaltă precizie pentru conversia temperatură ↔ tensiune.

Forma generală pentru TEMPERATURĂ → TENSIUNE:

  V(T) = c₀ + c₁·T + c₂·T² + c₃·T³ + ... + cₙ·Tⁿ

Unde:
  V = Tensiune în milivolt (mV)
  T = Temperatura în grade Celsius (°C)
  c₀, c₁, ..., cₙ = Coeficienți polinomiali specifici tipului

Pentru Type K (0°C to 1372°C):
  V(T) = c₀ + c₁·T + c₂·T² + ... + c₉·T⁹ + a₀·exp[a₁(T-a₂)²]

  Coeficienți (primii 5):
    c₀ = -0.176004136860 x 10⁻¹
    c₁ =  0.389212049750 x 10⁻¹
    c₂ =  0.185587700320 x 10⁻⁴
    c₃ = -0.994575928740 x 10⁻⁷
    c₄ =  0.318409457190 x 10⁻⁹
    ...plus termen exponențial


12.2 COMPENSAREA JONCȚIUNII RECI
----------------------------------
Termocuplele măsoară diferența de temperatură între cele două joncțiuni.
Pentru a obține temperatura absolută, trebuie compensată temperatura
joncțiunii reci (Cold Junction Compensation - CJC).

Temperatura măsurată = Temperatura calculată din V + Temperatura joncțiune rece

Metode de CJC:
1. Baie de gheață la 0°C (laborator)
2. Senzor RTD/termistor la joncțiune (industrial)
3. Compensare electronică (circuite specializate)

În aplicația RTD Master Pro:
- Utilizatorul poate seta manual temperatura joncțiunii reci
- Valoare implicită: 25°C (temperatura camerei)
- Calcul automat cu compensare


12.3 CALCUL INVERS (TENSIUNE → TEMPERATURĂ)
---------------------------------------------
Pentru conversia inversă, se folosește metoda Newton-Raphson pentru a rezolva
ecuația polinomială iterativ.

Algoritmul:
1. Compensare CJC: V_compensat = V_măsurat + V(T_joncțiune_rece)
2. Estimare inițială: T₀ = 0°C
3. Iterare Newton-Raphson:
   T_(n+1) = T_n - [V(T_n) - V_compensat] / [dV/dT(T_n)]
4. Repetă până la convergență (|T_(n+1) - T_n| < toleranță)


13. UTILIZARE TERMOCUPLE ÎN RTD MASTER PRO
-------------------------------------------

13.1 SELECTARE MOD TERMOCUPLA
-------------------------------
1. Deschide RTD Master Pro
2. În secțiunea "Measurement Mode", selectează "Thermocouple"
3. Interfața se va schimba pentru a afișa controale specifice termocuplei

13.2 CONFIGURARE TERMOCUPLA
-----------------------------
1. Type: Selectează tipul termocuplei (K, J, T, E, N, S, R, B)
2. Range: Afișat automat pentru tipul selectat
3. Sensitivity: Afișat automat (µV/°C)
4. Cold Junction Temp: Setează temperatura joncțiunii reci (implicit 25°C)

13.3 CONVERSII SUPORTATE
-------------------------

MOD 1: TEMPERATURĂ → TENSIUNE
  Input: Temperatura în °C
  Output: Tensiune în mV
  Exemplu Type K:
    500°C → 20.644 mV

MOD 2: TENSIUNE → TEMPERATURĂ
  Input: Tensiune în mV
  Output: Temperatura în °C (cu compensare CJC)
  Exemplu Type K:
    20.644 mV (CJC=25°C) → ~500°C

----------------------

  - Input: 35.2 mV, CJC: 23°C
  - Output: ~852°C
