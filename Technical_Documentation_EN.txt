1. OVERVIEW
================================================================================

RTD Master Pro is a professional-grade desktop application designed for accurate
temperature measurements using both RTD (Resistance Temperature Detector) sensors
and thermocouples, according to multiple international standards.

RTD SENSORS:
Precision temperature measurement devices that exploit the predictable change
in electrical resistance of pure metals with temperature.

THERMOCOUPLES:
Temperature sensors that generate electrical voltage based on the Seebeck effect,
ideal for extreme temperatures and fast response.

The application implements standardized equations and coefficients defined by
international metrology organizations (IEC, NIST, ASTM, GOST).

RTD sensors are precision temperature measurement devices that exploit the
predictable change in electrical resistance of pure metals with temperature.
This application implements the standardized equations and coefficients
defined by international metrology organizations.

4. CALCULATION METHODS AND EQUATIONS
================================================================================

4.1 PLATINUM RTD CALCULATION - Callendar-Van Dusen Equation
-------------------------------------------------------------

The Callendar-Van Dusen equation is the internationally accepted method for
calculating platinum RTD resistance as a function of temperature.

For T ≥ 0°C:
  R(T) = R₀(1 + AT + BT²)

For T < 0°C:
  R(T) = R₀[1 + AT + BT² + C(T - 100)T³]

Where:
  R(T) = Resistance at temperature T (in Ω)
  R₀   = Nominal resistance at 0°C (in Ω)
  T    = Temperature (in °C)
  A, B, C = Callendar-Van Dusen coefficients

COEFFICIENT VALUES:

Pt385 (IEC 60751, ASTM E1137):
  A =  3.9083 × 10⁻³  °C⁻¹
  B = -5.775  × 10⁻⁷  °C⁻²
  C = -4.183  × 10⁻¹² °C⁻⁴

Pt391 (GOST 6651):
  A =  3.96847 × 10⁻³  °C⁻¹
  B = -5.847   × 10⁻⁷  °C⁻²
  C = -4.35    × 10⁻¹² °C⁻⁴

Pt392 (Legacy US):
  A =  3.9848 × 10⁻³  °C⁻¹
  B = -5.870  × 10⁻⁷  °C⁻²
  C = -4.40   × 10⁻¹² °C⁻⁴


INVERSE CALCULATION (Resistance → Temperature):

For R ≥ R₀ (T ≥ 0°C):
  Solve quadratic equation: BT² + AT + (1 - R/R₀) = 0

  T = (-A + √(A² - 4B(1 - R/R₀))) / (2B)

For R < R₀ (T < 0°C):
  Solve cubic equation iteratively using numerical methods (Newton-Raphson)


4.2 COPPER AND NICKEL RTD CALCULATION - Linear Approximation
--------------------------------------------------------------

Copper and nickel RTDs use a simplified linear relationship due to their
narrower temperature ranges and different physical properties.

Forward Calculation (Temperature → Resistance):
  R(T) = R₀(1 + αT)

Inverse Calculation (Resistance → Temperature):
  T = (R - R₀) / (R₀ × α)

Where:
  R(T) = Resistance at temperature T (in Ω)
  R₀   = Nominal resistance at 0°C (in Ω)
  α    = Temperature coefficient of resistance (in Ω/Ω/°C)
  T    = Temperature (in °C)


COPPER COEFFICIENTS:
  α₄₂₆ = 0.00426 Ω/Ω/°C
  α₄₂₇ = 0.00427 Ω/Ω/°C
  α₄₂₈ = 0.00428 Ω/Ω/°C

NICKEL COEFFICIENTS:
  α₆₁₇ = 0.00617 Ω/Ω/°C (Ni100)
  α₆₁₈ = 0.00618 Ω/Ω/°C (Ni1000)
  α₆₇₂ = 0.00672 Ω/Ω/°C (Ni120)

10. THERMOCOUPLES - GENERAL OVERVIEW
-------------------------------------

10.1 WHAT ARE THERMOCOUPLES?
-----------------------------
A thermocouple is a temperature sensor that generates an electrical voltage
(thermoelectric electromotive force) proportional to the temperature difference
between two junctions of dissimilar metals. It is based on the Seebeck effect,
discovered in 1821 by Thomas Johann Seebeck.

Operating principle:
- Two wires of different metals are joined at one end (measuring junction)
- The other end is at a reference temperature (cold junction)
- The temperature difference generates a small voltage (µV-mV)
- The voltage is directly proportional to the measured temperature

------------------------------------

12. THERMOCOUPLE CALCULATIONS AND EQUATIONS
--------------------------------------------

12.1 NIST ITS-90 POLYNOMIALS
-----------------------------
The NIST (National Institute of Standards and Technology) standard defines
high-precision polynomials for temperature ↔ voltage conversion.

General form for TEMPERATURE → VOLTAGE:

  V(T) = c₀ + c₁·T + c₂·T² + c₃·T³ + ... + cₙ·Tⁿ

Where:
  V = Voltage in millivolts (mV)
  T = Temperature in degrees Celsius (°C)
  c₀, c₁, ..., cₙ = Type-specific polynomial coefficients

For Type K (0°C to 1372°C):
  V(T) = c₀ + c₁·T + c₂·T² + ... + c₉·T⁹ + a₀·exp[a₁(T-a₂)²]

  Coefficients (first 5):
    c₀ = -0.176004136860 x 10⁻¹
    c₁ =  0.389212049750 x 10⁻¹
    c₂ =  0.185587700320 x 10⁻⁴
    c₃ = -0.994575928740 x 10⁻⁷
    c₄ =  0.318409457190 x 10⁻⁹
    ...plus exponential term


12.2 COLD JUNCTION COMPENSATION
---------------------------------
Thermocouples measure the temperature difference between two junctions.
To obtain absolute temperature, the cold junction temperature must be
compensated (Cold Junction Compensation - CJC).

Measured temperature = Calculated temperature from V + Cold junction temperature

CJC methods:
1. Ice bath at 0°C (laboratory)
2. RTD/thermistor sensor at junction (industrial)
3. Electronic compensation (specialized circuits)

In RTD Master Pro:
- User can manually set cold junction temperature
- Default value: 25°C (room temperature)
- Automatic calculation with compensation


12.3 INVERSE CALCULATION (VOLTAGE → TEMPERATURE)
-------------------------------------------------
For inverse conversion, the Newton-Raphson method is used to solve
the polynomial equation iteratively.

Algorithm:
1. CJC compensation: V_compensated = V_measured + V(T_cold_junction)
2. Initial estimate: T₀ = 0°C
3. Newton-Raphson iteration:
   T_(n+1) = T_n - [V(T_n) - V_compensated] / [dV/dT(T_n)]
4. Repeat until convergence (|T_(n+1) - T_n| < tolerance)
